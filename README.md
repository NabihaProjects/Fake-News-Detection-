Fake News Detection System

An AI-powered Fake News Detection platform that uses NLP and Transformer models to classify news articles as Fake or Real. The system also provides explainability features, confidence scores, and browser extension support for real-world usage.

Features
Fake vs Real news classification
Confidence score prediction
NLP preprocessing pipeline
Baseline ML model (TF-IDF + Logistic Regression)
Transformer-based model (DistilBERT/BERT)
Explainable AI predictions
FastAPI backend
React frontend
Chrome extension support
REST API integration
Deployment-ready architecture
